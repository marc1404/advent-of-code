# Advent of Code 2019

## Requirements
- [Node.js](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/lang/en/)

## Scripts
- `yarn dev <day>` - Development mode
- `yarn start <day>` - Production mode
- `yarn day1` - Day 1
- `yarn day2` - Day 2
- `yarn day3` - Day 3
- `yarn day4` - Day 4
- `yarn day5` - Day 5
- `yarn day6` - Day 6
- `yarn day7` - Day 7
- `yarn day8` - Day 8
- `yarn day9` - Day 9
- `yarn day10` - Day 10
